ForgetMeNot ist eine App zum Lernen mithilfe von Karteikarten. Einfachheit, Benutzerfreundlichkeit und Geschwindigkeit wurden bei der Entwicklung dieses Bildungsprogramms berücksichtigt. Diese App bietet hervorragende Bedingungen, um eine hohe Lerneffizienz zu erzielen.


Die unterstützten Funktionen sind:

    • Importieren/Exportieren von Dateien.
    • Unterstützt CSV, Tabtext oder jede andere Art von Trennzeichen.
    • Intervalle (Verteilte Wiederholung). Sie können für jedes Deck ein eigenes Intervallschema festlegen. Es ist auch möglich, einen eigenen Algorithmus anzulegen, wie die Antworten sich auf das nächste Intervall auswirken.
    • Verschiedene Testmethoden. Es gibt 'Selbsttest', 'Testen mit Varianten' und 'Rechtschreibprüfung'.
    • Aussprache des Textes über TTS. Sie können Sprachen für Fragen und Antworten auswählen und das automatische Vorsprechen von ihnen aktivieren.
    • Verstecken Sie den Text einer Frage zur Verbesserung der Fähigkeiten des Hörens, dies ist sehr nützlich für das Erlernen von Fremdsprachen.
    • Karten umkehren.
    • Hinweise in Form von Hervorhebung von Buchstaben.
    • 'Motivations-Timer', mit dem Sie sich auf Ihr Studium konzentrieren können (optional).
    • Speichern von Voreinstellungen von Stapeln um sie wiederzuverwenden und Wiederholungen bei Einstellungen zu vermeiden.
    • Bearbeiten und Suchen von Karten direkt in der Übung.
    • 'Laufmodus', der es Ihnen ermöglicht, die Übung ohne einen Blick auf den Bildschirm durchzuführen.
    • 'Automatischer Modus'. In diesem Modus werden Fragen und Antworten nacheinander ausgesprochen. Sie können Ihre eigenen Aktivitäten und Wiederholungen von Unterrichtsmaterial kombinieren.
    • Katalog von vorgefertigten Stapeln. Der Katalog enthält viele Stapel zum Sprachenlernen, die eine Basis von Wörtern, thematischen Wörtern und Phrasen, und ganzen Sätzen enthalten.
    • Gruppieren von Stapeln in getrennte Listen.
    • Anpassen der Kartenansicht.
    • Dunkles Farbschema.


Was ForgetMeNot so effizient für das Einprägen macht:

    - Keine Werbung. Sie können eine Menge Karten in kurzer Zeit bearbeiten, weil das Interface so einfach und bequem ist.  Nichts hindert Sie daran, zu studieren.
    - Die Übung ist so konzipiert, dass man irgendwann die Antwort verstehen muss. Wenn Ihre Antwort falsch ist, wird die Karte ans Ende der Liste verschoben, bis Sie richtig antworten.
    - Intervalle (Verteilte Wiederholung). Die verteilte Wiederholung reduziert das Vergessen, das auftritt, wenn Material für einen längeren Zeitraum nicht gesichtet wird. Aktives erinnern von gelernten Material ist auch enthalten, welches lernen begünstigt.
    - Möglichkeit eine Karte direkt in der Übung zu bearbeiten. Dadurch können Sie nicht nur Fehler korrigieren, sondern auch Karten neu erstellen, wenn sie schwer zu merken sind. Zum Beispiel können Sie eine zusätzliche Assoziation in Klammern hinzufügen.
    - Sprachbegleitung. Dies ist besonders hilfreich beim Erlernen von Sprachen, da es das Hörverständnis verbessert, man sich auf das Lernen konzentriert und die Erinnerung verbessert. Sie können sogar einen Stapel einrichten, um eine Frage ausschließlich durch das Ohr wahrzunehmen.
    - Modi, mit denen Sie nicht nur in Ihrer Freizeit üben können. Der "Lauf-Modus" ermöglicht es dir, die Übung während des Laufens durchzuführen. 'Automatischer Modus' erlaubt es dir, Fragen und Antworten anzuhören, wenn deine Hände und Augen beschäftigt sind, aber deine Ohren frei sind : )
    - Fokussierende Hilfsmittel. Der 'Motivationstimer' erinnert Sie daran, wenn Sie Ihre Aufmerksamkeit verloren haben. Der Vollbildmodus verbirgt die Statusleiste, was eine zusätzliche Ablenkungsquelle sein könnte.
